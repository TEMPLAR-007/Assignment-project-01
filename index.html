Mission

Learn the basic concepts of HTML, CSS and JavaScript to create web pages and web applications. Here are some expert-led, beginner courses that will start with the basics of creating a web page and styling it all the way up to creating interaction with JavaScript.

Vission

Web developers learn by doing. Set up a Github account and push code to the site. Practicing coding and getting used to using Git prepares developers to work as part of a team, as well as manage complicated site and application builds.

Goals

Employers want to see results, such as current examples of successful site and application builds. With a portfolio, developers can demonstrate their coding talent and technical capabilities. Standout portfolios can help candidates clear that crucial first hurdle: landing a job interview.
